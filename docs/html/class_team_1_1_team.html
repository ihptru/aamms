Respawns the whole team.

This function respawns every player in the team.

Parameters:
xThe x-coordinate where to spawn to player with the position 1 in the team.
yThe y-coordinate where to spawn to player with the position 1 in the team.
xdirThe x direction
ydirThe y direction
offsetThe offset for back-moving the players.
shiftThe offset for moving the players left or right. Should be greater than 0
forceForce the new position for each player? True if yes.
Attention:
x and y coordinate are NOT relative to xdir and ydir.
Note:
This triggers the Event "Team spawned""
