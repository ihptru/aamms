Sets the team of the player.

This function lets the player join the given team

Parameters:
teamname The escaped name of the team to which to add the player. None if the player is specating. AI if the player is in the AI team.
force Create the team if it doesn't exist?
quiet If True, don't print info log messages
Exceptions:
RuntimeError Raised if the team doesn't exist and force isn't set
Note:
This also calls Team.addPlayer (adds the player to the team)
This triggers the event "Player joined team"
If the team name is AI, is_human is set to False
