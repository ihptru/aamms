  teamname,
  force = False,
  quiet = False 
)

Sets the team of the player.

This function lets the player join the given team

Parameters:
teamnameThe escaped name of the team to which to add the player. None if the player is specating. AI if the player is in the AI team.
forceCreate the team if it doesn't exist?
quietIf True, don't print info log messages
Exceptions:
RuntimeErrorRaised if the team doesn't exist and force isn't set
Note:
This also calls Team::addPlayer (adds the player to the team)
If the team name is AI, is_human is set to False

)

Set player's lives.

This sets the player's lives to the given number. A value less or equal 0 means the player is death.

Parameters:
livesThe lives
Note:
If the number of lives is less than 0, 0 is used for lives.
Setting the lives to 0 does NOT immediately kill the player. For that, use Player::kill.
