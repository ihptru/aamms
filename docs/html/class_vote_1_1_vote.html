)

Checks the result of the vote.

Checks if the vote successed or failed and triggers "Vote successed" or "Vote failed" event.

Parameters:
not_voted What happens with players who didn't vote? Could be "yes", "no" or "dont_count". If "yes"|"no" is given, players who didn't vote are counted as if the had voted "yes"|"no". Otherwise, they aren't counted. Default is "dont_count".
not_voted_spec Same as not_voted, but only for spectators. If not given or set to None, not_voted is used.
min_needed How much precent yes votes are at least needed that the vote gets accepted? Default 51.
Exceptions:
ValueError Raised if not_voted ot not_voted_spec aren't on of "yes", "no" or "dont_count".
